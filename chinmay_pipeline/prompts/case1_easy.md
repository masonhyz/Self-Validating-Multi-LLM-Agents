# Number String
Generate a single string of exactly 20 decimal digits (0–9) with all of the following constraints. It needs to have the following verifiable constraints 

## VERIFIABLE CONSTRAINTS 
- The string contains exactly 1 occurrence of the substring 2026
- The string contains no occurrence of 000
- The string must not start with 0
- The string must end with an odd digit
- The total number of even digits (0,2,4,6,8) is exactly 10
